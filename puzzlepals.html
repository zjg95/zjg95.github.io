what was done

I created a simple 2D game in java modeled after the existing game Pokemon Puzzle League.

when was it done

November - December 2013

where was it done

Austin, TX

why was it done

To learn more about programming techniques and become familiar with Java's GUI libraries.

how was it done

I used Java, and its Swing and graphics libraries.

description

PuzzlePals is a simple 2D puzzle video game that I created after completing my first programming course in Java. I modeled the game after an existing puzzle game style. The game supports two players whose actions actively affect each other’s play environment. I spent just about all my free time during the fall and spring semesters working on this project.

Since I was a novice at the time, I encountered many challenges during this project that I didn’t understand, such as synchronization with multiple threads of execution. Though I later learned how to implement these solutions in my upper division coursework, I did not return to PuzzlePals to update the code.
